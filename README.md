## 📄 Project Overview

**Casa Movil Sur** is a professional real estate platform dedicated to the sale and management of mobile homes and land in the Costa del Sol region, Málaga, Spain. The website provides a comprehensive showcase of available properties, specifications, and a direct contact system for potential buyers.

### ✨ Key Features

- 🏠 **Property Showcase**: Interactive carousels displaying various mobile home models.
- 📍 **Location Highlights**: Detailed information about the benefits of living in Costa del Sol.
- 🛠️ **Detailed Specifications**: Tabbed interface for exploring different rooms and features of the homes.
- 📸 **Photo Gallery**: High-quality image gallery with lightbox support.
- 📩 **Contact System**: AJAX-powered contact form for direct inquiries.
- 📱 **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- 💬 **WhatsApp Integration**: Quick access button for instant communication.

## 🚀 Installation & Local Development

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/joelmarquez/casamovilsur.git
   ```

2. **Server Requirements**:
   Since the contact form uses PHP, you need a local server environment like **XAMPP**, **WAMP**, or **Laragon**.

3. **Setup**:
   - Copy the project folder to your server's root directory (e.g., `C:/xampp/htdocs/`).
   - Start **Apache** from your server control panel.

4. **Access**:
   Open your browser and navigate to:
   ```text
   http://localhost/casamovilsur
   ```
